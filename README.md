🎵 PrimeSound - Backend (Node.js & MongoDB)
📝 Giới thiệu
PrimeSound Backend là hệ thống cung cấp API cho nền tảng nghe nhạc trực tuyến. Dự án tập trung vào việc xử lý dữ liệu âm nhạc lớn, quản lý danh sách phát và tối ưu hóa tốc độ stream nhạc.

👨‍💼 Quản trị & Điều phối (Project Manager Perspective)
Mô hình: Agile/Scrum với các kỳ Sprint 2 tuần.
Vai trò: Project Manager & Tech Lead.
Chiến lược dữ liệu: Lựa chọn MongoDB (NoSQL) để tận dụng cấu trúc Schema linh hoạt, giúp team dễ dàng mở rộng các thuộc tính của Metadata nhạc và Playlist mà không gây gián đoạn hệ thống.
Kiểm soát chất lượng: Trực tiếp thiết kế cấu trúc thư mục chuẩn RESTful API và triển khai Automated Testing để đảm bảo tính ổn định của các dịch vụ cốt lõi.

🚀 Tính năng Backend
[x] Music Streaming API: Xử lý luồng dữ liệu âm nhạc chất lượng cao.

[x] Flexible Metadata: Quản lý thông tin bài hát, nghệ sĩ, album qua MongoDB Schema.

[x] Playlist Management: Logic xử lý thêm/xóa/sắp xếp bài hát linh hoạt.

🛠 Hướng dẫn cài đặt
Cài đặt Node.js và MongoDB.
Cấu hình chuỗi kết nối (Connection String) trong file .env.

Chạy lệnh:
npm install
npm start
